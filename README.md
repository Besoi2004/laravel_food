# Hệ Thống Quản Lý Nhà Hàng

## Giới thiệu
Hệ thống quản lý nhà hàng được xây dựng bằng Laravel Framework, cung cấp giải pháp toàn diện cho việc quản lý menu, đơn hàng và báo cáo.

## Tính năng chính

### 1. Quản lý Menu
- Thêm, sửa, xóa món ăn
- Phân loại theo danh mục:
  - Món chính (Main Dish)
  - Món khai vị (Appetizer)
  - Món tráng miệng (Dessert)
  - Đồ uống (Beverage)
- Tìm kiếm món ăn nâng cao
- Quản lý giá và mô tả

### 2. Quản lý Đơn Hàng
- Theo dõi đơn hàng theo thời gian thực
- Xử lý đơn hàng
- Lịch sử đơn hàng
- Chi tiết đơn hàng

### 3. Dashboard Admin
- Thống kê doanh thu
- Báo cáo đơn hàng
- Quản lý người dùng

### 4. Bảo Mật
- Hệ thống đăng nhập an toàn
- Phân quyền admin
- Bảo vệ thông tin khách hàng

## Công nghệ sử dụng
- Laravel 9.x
- MySQL
- Bootstrap 5
- JavaScript/jQuery
- AJAX
- SweetAlert2
